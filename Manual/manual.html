Brah Rah's Save Some Seconds

Addon Overview

This addon saves time working with Godots nodes and editors like visual studio code/codium etc. It also makes working with the c# .net diconnect/failed to unload assembiles a bit more compfortable.

Pre-Requisite Setup:

1. You need to use the c# version of godot (Godot Engine - .net):

2. Extract the archive file BrsSSS.zip into your Godot project's folder.

3. Create C# solution (If you haven't already):

4. Build Project (Alt+B):

5. Enable the addon in Project - Project Settings - Plugins:




















Detailed Breakdown:

Node to Code:

This feature generates code using the selected node. There are three Selected Node To: buttons, each linked to a corresponding TextEdit field below it.
To use this, write your code and use the following placeholders:
<<NodeName>> replaces it with the selected node's name.
<<NodeType>> replaces it with the selected node's type.
<<NodePath>> replaces it with the selected node's path.

For example, if you enter:
public <<NodeType>> <<NodeName>> = <<NodePath>>; into one of the TextEdits, select a Label node, and press Selected Node To:, it will be converted into:
Label YourSelectedLabelsName = SomeContainer/YourSelectedLabelsName;
The result is copied to the clipboard, ready to be pasted into your code editor.

Tip: You can use both C#, GDScript.

Detect and Set Unique
This feature detects occurrences of GetNode or get_node in your code and, when enabled, sets the selected node to "Unique." (Equivalent to: Right-click on a node â†’ Access as Unique Name).


Set Node1 and Get Relative Path
Retrieves the relative path between two nodes and copies it to the clipboard. Usage: - Select a node and press Set Node1 to set it as the starting point. - Then, select the destination node and press Get Relative Path.

Restart Editor
Restarts the Godot editor.

Del GDtemp
Deletes the .godot cache folder. Useful for resolving certain errors produced by Godot.


Addons:
Enables, disables, or restarts addons listed in the TextEdit field.
Syntax: "AddonName", "Another Addon's Name", "etc." Anything inside quotation marks is recognized as an addon and will be affected by the buttons.




Open Folders:
User Opens the user folder (user://) in the system's file browser.
Project Opens the current project's folder (res://).


Open Manual
Opens the manual in the system's default web browser.



About the .NET Disconnect Issue:

This section raises awareness about an ongoing .NET disconnect issue. Since Godot adopted .NET in version 4, this problem has persisted, making some addons nearly unusable.
Pressing Build Project (Alt+B) disconnects .NET, requiring a restart of Godot for C# to work again.
Spreading awareness might help get it fixed.
If you've read this already, you can dismiss it by clicking Got It!.
